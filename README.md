# LinkIt jQuery Plugin
**LinkIt** is a simple jQuery plugin to attach links to elements
## Version
0.4.0
## Usage
```javascript
    $('span').linkIt({
      href: 'http://test.com',
      text: 'Click Me',
      target: "_blank"
    });
```
